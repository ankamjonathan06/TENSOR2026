# 🚀 AdaptZip – AI-Powered Adaptive Compression Engine

> "Compress smarter. Archive better. Let science scale without limits."

## Overview

AdaptZip is an AI-powered adaptive compression system that analyzes scientific datasets at the column level, selects optimal compression algorithms dynamically, and ensures reconstruction fidelity through RMSE validation and lossless verification.

## Architecture

- **Frontend**: React.js + Tailwind CSS + Recharts (Vite)
- **Backend**: Node.js + Express.js
- **AI Engine**: Integrated ML-based algorithm selection
- **Database**: MongoDB Atlas

## Quick Start

```bash
# Install dependencies
cd backend && npm install
cd ../frontend && npm install

# Run backend
cd backend && npm run dev

# Run frontend (new terminal)
cd frontend && npm run dev
```

## Features

- 📊 Statistical column profiling (entropy, sparsity, distributions)
- 🧠 AI-based algorithm selection per column
- ⚡ Parallel compression with >100 MB/s throughput
- ✅ RMSE validation + lossless verification
- 📈 Real-time dashboard with compression analytics
- 📋 Explainable algorithm selection reports

## Supported Formats

- CSV
- HDF5
- NetCDF
